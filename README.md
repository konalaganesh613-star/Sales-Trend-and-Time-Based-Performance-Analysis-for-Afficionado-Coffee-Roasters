# Sales-Trend-and-Time-Based-Performance-Analysis-for-Afficionado-Coffee-Roasters# ☕ Coffee Sales Trend & Performance Analysis

 Project Overview
This project analyzes transactional sales data from Afficionado Coffee Roasters to identify daily, weekly, and hourly sales trends. The objective is to uncover time-based demand patterns and provide actionable business insights for operational planning and decision-making.

The project includes:
- Data Cleaning & Feature Engineering (Python)
- Exploratory Data Analysis (EDA)
- Research Paper (PDF)
- Executive Summary (PDF)


 Tools & Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Streamlit
- Google Colab
- GitHub

---

 Dataset Description

The dataset contains the following columns:

- Transaction ID
- Transaction Time
- Product Name
- Store Location
- Transaction Quantity
- Unit Price

Additional engineered features:
- Revenue (Quantity × Unit Price)
- Hour of Transaction
- Day of Week
- Time Bucket (Morning, Afternoon, Evening, Late Night)

---

 Key Analysis Performed

- Data Cleaning (null removal, duplicate removal)
- Revenue calculation
- Time-based feature extraction
- Daily revenue trend analysis
- Day-of-week performance comparison
- Hourly peak analysis
- Store-level revenue comparison
- Heatmap visualization (Day vs Hour)

---

Key Insights

- Morning hours (7 AM – 10 AM) generate highest revenue.
- Fridays show peak weekly performance.
- Mid-week sales are comparatively lower.
- Certain store locations outperform others.
- Sales significantly decline during late-night hours.

---

 Business Recommendations

- Increase staffing during peak morning hours.
- Introduce promotional campaigns during slow mid-week periods.
- Optimize inventory based on time-of-day demand.
- Monitor underperforming locations for operational improvements.
